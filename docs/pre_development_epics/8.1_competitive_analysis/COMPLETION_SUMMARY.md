### What We Learned

   - Event RSVP: No competitor has this
   - Integrated registry: Completely unique
   - Role-based access: Unique to Nonna
   - These features are hard to copy quickly (defensible)

4. **Technology Stack is Strategic Advantage**

   - Better data model fit (PostgreSQL for relational data)
